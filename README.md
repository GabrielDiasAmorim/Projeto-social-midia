## 📌 Sobre o projeto

O **Projeto Social Mídia** é uma página web simples que simula um agregador de links de redes sociais (estilo Linktree), permitindo centralizar perfis em um único lugar.

Este projeto foi desenvolvido durante o curso de HTML e CSS do Curso em Vídeo, com foco em praticar conceitos fundamentais de desenvolvimento web.

---

## 🛠️ Tecnologias utilizadas

* HTML5
* CSS3

---

## 💻 Funcionalidades

* Exibição de perfil
* Links para redes sociais
* Layout simples e organizado
* Interface limpa e intuitiva

---

## 📱 Responsividade

Este projeto foi desenvolvido com foco inicial em aprendizado, podendo não estar totalmente responsivo para todos os dispositivos.

---

## 🚀 Aprendizados

Durante o desenvolvimento deste projeto, foram praticados:

* Estrutura básica do HTML
* Estilização com CSS
* Organização de elementos na tela
* Publicação com GitHub Pages
